# Uber Trip Analysis Dashboard (Excel)

## Project Overview
This project analyzes Uber trip data to understand booking trends, revenue performance,
customer behavior, and operational efficiency using Microsoft Excel.

## Business Objective
To identify:
- Peak booking hours
- Top-performing ride categories
- Revenue contribution by vehicle type and payment method
- Ride completion and cancellation patterns

## Key KPIs
- Total Bookings
- Total Revenue
- Average Booking Value
- Average Ride Distance
- Average Customer Rating

## Dashboard Insights
- Monthly booking trend to track demand fluctuations
- Ride completion overview to analyze operational success
- Revenue by payment method to understand customer preferences
- Peak booking hours for demand forecasting
- Top pickup and drop locations for location-based planning
- Revenue per vehicle type to optimize fleet allocation

## Tools & Skills Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers for interactivity
- Data Cleaning & Formatting
- Business Analytics & Visualization
